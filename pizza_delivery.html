Pizza Delivery Excel Project

goal: creating a dynamic dashboard for the decision makers
tools used: Excel

Project details

Download project file

Here you can check out the original data set on kaggle.

The original dataset contains the yearly details of an imaginary pizza delivery shop in 4 different csv files. Each of the files has a few columns and the longest one has about 21 400 rows. It is quite big, but still manageable, so I decided to turn this into an Excel project.

  • the first 4 tabs hold the original csv files
  • created a working sheet where I cleaned the data
  • made 4 pivot tables answering the following questions:
    1. Which pizza category is generating the most income?
    2. Which size is ordered the most (also further divided by category)?
    3. Is there any connection of what size of pizza people order and the time of the day?
    4. What are the most popular pizzas?
  • I created a dashboard showing the 4 visualizations, also adding filters. You can filter for specific month of the year and specific pizza types where it makes sense.
